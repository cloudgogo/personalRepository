- 单点登录说明
在之前的实现方式是通过oa链接过来，通过get方式传递sso参数，代表链接是通过oa系统过来的，这么做的目的是需要保留系统的登录窗口以实现admin账户的登录。在链接到达报表平台时，通过filter解析传递过来的token解析出正确的用户名，进行登录。
